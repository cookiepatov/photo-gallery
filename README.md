# "Галерея"

### [перейти](https://cookiepatov.github.io/photo-gallery/)

## Описание

Сайт представляет собой личную фото- и видео-галерею автора. Все интерактивные элементы, а так же фон анимированы, есть возможность полноэкранного просмотра и дальнейшее переключение изображений. На странице с видео есть возможность смотреть клипы прямо с сайта. Сайт полностью адаптивен. Есть возможность переключать чёрно-белый и цетной режимы отображения.


## Используемые технологии и методологии

Сайт свёрстан по методологии БЭМ, используется grid подход.
